# Rock Paper Scissors Game

A classic Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer and track your score!

## Features

- **Interactive Gameplay**: Choose between Rock, Paper, or Scissors to play against the computer.
- **Score Tracking**: The game keeps track of your Wins, Losses, and Ties.
- **Persistent Score**: Scores are saved in the browser's local storage, so you don't lose your progress when you refresh the page.
- **Visual Feedback**: Displays the moves made by both you and the computer using images.
- **Reset Option**: A reset button allows you to clear the score and start fresh.

## How to Play

1. **Clone or Download** the repository to your computer.
2. Locate the project folder.
3. Open `rps.html` in your web browser.
4. Click on the **Rock**, **Paper**, or **Scissors** icons to make your move.
5. The result (Win/Lose/Tie) will be displayed instantly, and the score will update.

## Project Structure

- `rps.html`: The main structure of the game interface.
- `script.js`: Contains the game logic, including the computer's move generation, result calculation, and score management.
- `style.css`: Handles the styling and layout of the game.
- `images/`: Directory containing the images for Rock, Paper, and Scissors.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
